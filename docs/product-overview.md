# TimeMosaic - プロダクト概要

## 目的

TimeMosaic は、ローカルの Markdown メモファイルに記録した歴史イベントを、時系列で視覚的に表示するローカルホスト Web アプリケーションです。

## コンセプト

- **メモとの統合**: 既存のメモ管理ツール（Obsidian など）で管理している Markdown ファイルを直接参照
- **ローカルファースト**: すべてのデータはローカルファイルとして保存され、プライバシーが保護される
- **シンプルで直感的**: 複雑な設定不要で、Markdown を書くだけで年表が生成される
- **柔軟な粒度**: 年単位から日付単位まで、必要に応じた精度で記録可能

## ユースケース

1. **個人の歴史学習**: 興味のある時代の出来事を記録し、時系列で理解する
2. **プロジェクト履歴**: 開発プロジェクトのマイルストーンを記録し、振り返る
3. **人生の年表**: 自分や家族の重要な出来事を記録する
4. **研究ノート**: 研究対象の出来事を時系列で整理する

## 主要機能（MVP）

1. **時系列表示**: イベントを年表形式で表示
2. **フィルタリング/検索**: キーワードや期間で絞り込み
3. **タグ分類**: タグによるカテゴリ分けと絞り込み
4. **自動リロード**: メモファイル更新時に自動反映

## 非機能要件

- **パフォーマンス**: 数千件のイベントでも快適に表示
- **ユーザビリティ**: 直感的な UI で学習コスト最小化
- **保守性**: シンプルな技術スタックで長期的に保守可能
