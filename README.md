# TimeMosaic

複数の年表をモザイクのように並べて比較できる、ローカルファースト年表ビューア

## 特徴

- **ローカルファースト**: Markdown メモファイルを直接参照、データはすべてローカルに保存
- **マルチカラム表示**: 複数のタグを選択して年表を並べて比較
- **シンプルな記法**: Markdown で書くだけ、特別な設定不要
- **自動リロード**: メモファイルを編集すると即座にブラウザに反映
- **柔軟な日付**: 年のみ、年月、年月日まで対応

## クイックスタート

### 必要な環境

- Node.js 18 以上
- npm または pnpm

### インストール（未実装）

```bash
# リポジトリをクローン
git clone https://github.com/yourusername/TimeMosaic.git
cd TimeMosaic

# 依存関係をインストール
npm install

# 環境変数を設定
cp .env.example .env
# .env を編集してメモファイルのパスを設定

# 開発サーバーを起動
npm run dev
```

ブラウザで `http://localhost:5173` にアクセス

## メモファイルの書き方

```markdown
## 1945-08-15 終戦記念日
#歴史 #日本

第二次世界大戦の終結を告げる玉音放送が流れた日。

---

## 1969 アポロ11号月面着陸
#科学 #宇宙

人類初の月面着陸に成功。

---

## 2024 TimeMosaic プロジェクト開始
#個人 #開発

歴史イベントを視覚化する Web アプリケーション開発開始。
```

**フォーマット:**
1. `## YYYY-MM-DD イベント名` または `## YYYY イベント名`
2. 次行に `#タグ1 #タグ2 ...`（最低1つ必須）
3. その後に説明文（任意）
4. イベント間は `---` で区切る（推奨）

詳細は [データフォーマット仕様](./docs/data-format-spec.md) を参照

## 使い方

1. **タグを選択**: 上部のタグ選択 UI から興味のあるタグを選択
2. **年表が並ぶ**: 選択したタグごとに列が表示される
3. **比較**: 複数の年表を並べて時代の流れを比較
4. **検索**: 検索バーでキーワード検索
5. **メモ編集**: メモアプリで Markdown を編集すると自動で反映

## コンセプト

**TimeMosaic** = Time (時間) + Mosaic (モザイク)

複数の年表をモザイクのように配置することで、異なる視点から歴史の流れを理解できるツールです。

例えば「日本の歴史」と「西洋美術の歴史」を並べて表示することで、同時代の出来事の関連性や文化的背景を発見できます。
